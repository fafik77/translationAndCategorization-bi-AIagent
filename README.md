# Translation And Categorization AI Project

This project aims to explore and familiarize us with AI and LLM creation methods.

## Pre Requirements

- [Ollama](https://ollama.com/)
- Python3 (preferably as venv -virtual environment eg. [Miniconda ui](https://www.anaconda.com/docs/getting-started/miniconda/main))

### How to open this in VSCode with Miniconda

After importing this repo as workspace to VSCode:

- (Ctrl+Shift+P) -> "Python: Select Interpreter" -> "base (3.**)"
- Instal the requirements, in terminal (Ctrl+J or Ctrl+Shift+\`) type `pip install -r ./project/requirements.txt`
